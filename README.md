DevOne 3D Portfolio

An interactive 3D portfolio website built inside a virtual laptop using Three.js, HTML, CSS, and JavaScript.
Visitors can explore my projects in a 3D environment that blends design and technology.

Features
		Interactive 3D laptop built with Three.js
		Real website content rendered on the laptop screen
		Smooth camera movement and zoom controls

Inspiration

Most portfolio websites feel static and predictable. I wanted to build something more engaging.
This idea turned into a 3D world where the portfolio itself exists inside a virtual laptop to showcase my work.
